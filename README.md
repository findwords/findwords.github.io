# Find English Words Website

## Overview

This repository contains the source code for the **Find English Words** website, a tool designed to help users discover and learn English words. The website is hosted at [https://findwords.github.io/](https://findwords.github.io/) and provides a simple, user-friendly interface for word exploration.
